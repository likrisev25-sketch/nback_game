# ✅ РАБОЧАЯ ВЕРСИЯ ИГРЫ

## 🚀 Быстрый старт

### 1. Запуск
```bash
cd nback-game
npm run dev
```

### 2. Открыть игру
Откройте в браузере: **http://localhost:3000/play**

## 🎮 Как играть

### Настройки игры
- **N-значение** (1-4): Сложность игры. N-back означает, что нужно сравнивать с позицией N шагов назад
- **Шаги** (10-50): Количество ходов в игре
- **Скорость** (1000-3000мс): Время показа каждой позиции
- **Играть с ботом**: Включить/выключить бота
- **Точность бота** (50-100%): Насколько хорошо играет бот

### Правила игры
1. На сетке 3x3 подсвечиваются позиции по одной
2. Нажимайте "Совпадение!" если **текущая позиция** совпадает с позицией **N шагов назад**
3. Игра заканчивается после всех шагов
4. Показываются результаты: ваши и бота

### Пример (2-back)
```
Шаг 1: Позиция 0  → Нет совпадений (меньше 2 шагов)
Шаг 2: Позиция 5  → Нет совпадений (меньше 2 шагов)
Шаг 3: Позиция 0  → СОВПАДЕНИЕ! (позиция 0 была на шаге 1)
Шаг 4: Позиция 3  → Нет совпадений (позиция 3 не была на шаге 2)
Шаг 5: Позиция 5  → СОВПАДЕНИЕ! (позиция 5 была на шаге 3... нет, на шаге 2)
```

## 🎯 Советы

### Для начала
1. Начните с **1-back** (самый лёгкий)
2. **20 шагов** и **2000мс** (нормальная скорость)
3. Включите **бота** чтобы увидеть как играть

### Для прогресса
- Увеличивайте N постепенно (1 → 2 → 3)
- Уменьшайте скорость (2000 → 1500 → 1000)
- Увеличивайте количество шагов

## 🐛 Если что-то не работает

### Игра не загружается
```bash
# Перезапустите сервер
npm run dev

# Очистите кэш
rm -rf .next
npm run dev
```

### Ошибки в консоли
- Проверьте что сервер запущен
- Проверьте что открыли http://localhost:3000/play
- Проверьте консоль браузера (F12)

### Бот не играет
- Убедитесь что поставили галочку "Играть с ботом"
- Бот отвечает с задержкой 500-1000мс
