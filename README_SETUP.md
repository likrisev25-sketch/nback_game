# 🎮 N-Back Game - Руководство по запуску

## Быстрый старт

### 1. Установка зависимостей

```bash
cd nback-game
npm install
```

### 2. Настройка базы данных

```bash
# Создание таблиц для лобби
npm run db:setup-lobby

# Создание таблиц для турниров
npm run db:setup-tournament
```

### 3. Запуск в режиме разработки

```bash
npm run dev
```

Откройте http://localhost:3000

## Система лобби

### Создание лобби

1. Перейдите на `/lobbies`
2. Нажмите "+ Создать лобби"
3. Заполните форму:
   - Название (опционально)
   - Минимум игроков (2-6)
   - Максимум игроков (2-6)
   - N-back значение (1-4)
   - Скорость (1000ms-3000ms)
   - Пароль (опционально)
4. Нажмите "Создать"
5. Выберите стратегию запуска:
   - **Начать игру сразу** - ручной запуск
   - **Включить автозапуск** - игра начнётся автоматически когда все игроки нажмут "Готов"

### Игра в лобби

1. После запуска лобби статус меняется на "in_progress"
2. Показывается компонент игры N-back
3. Игрок видит:
   - Сетку 3x3
   - Кнопку "Совпадение"
   - Счёт правильных ответов и ошибок
   - Текущую скорость
   - N-значение
4. Нажмите "Совпадение", если текущая позиция совпадает с позицией N шагов назад
5. После завершения игры возвращаетесь в лобби

## Система турниров

### Создание турнира

1. Перейдите на `/tournaments`
2. Нажмите "+ Создать турнир"
3. Заполните форму:
   - Название
   - N-значение (1-4)
   - Раундов (3/5/10)
   - Шагов в раунде (20/30/50)
   - Скорость (1500ms-2500ms)
   - Минимум/максимум игроков (2-6)
   - Пароль (опционально)
4. Нажмите "Создать турнир"

### Присоединение к турниру

1. Перейдите на `/tournaments`
2. Найдите турнир в списке
3. Нажмите "Присоединиться"
4. На странице турнира нажмите "Готов"
5. Хост нажимает "Начать турнир"
6. Начинается обратный отсчет (5 секунд)

## API Endpoints

### Лобби
- `GET /api/lobby/list` - список всех лобби
- `POST /api/lobby/create` - создать лобби
- `GET /api/lobby/[id]` - данные лобби
- `POST /api/lobby/[id]/join` - присоединиться
- `POST /api/lobby/[id]/leave` - выйти
- `POST /api/lobby/[id]/ready` - готов/не готов
- `POST /api/lobby/[id]/start` - начать игру

### Турниры
- `GET /api/tournament/list` - список турниров
- `POST /api/tournament/create` - создать турнир
- `POST /api/tournament/join` - присоединиться
- `GET /api/tournament/[id]` - данные турнира

### Игры
- `POST /api/game/create` - создать игру
- `GET /api/game/list` - список игр
- `POST /api/game/join` - присоединиться
- `GET /api/game/[id]/start` - начать игру
- `GET /api/game/[id]/sequence` - получить последовательность
- `POST /api/game/[id]/answer` - отправить ответ
- `GET /api/game/[id]/stats` - статистика

## WebSocket Events

### Лобби
- `lobby:join` - присоединение
- `lobby:leave` - выход
- `lobby:ready` - готовность
- `lobby:start` - запуск
- `lobby:update` - обновление
- `lobby:player-joined` - игрок присоединился
- `lobby:player-left` - игрок покинул
- `lobby:player-ready` - готовность изменена
- `lobby:countdown` - обратный отсчет
- `lobby:start-game` - запуск игры

### Турниры
- `tournament:join` - присоединение
- `tournament:leave` - выход
- `tournament:ready` - готовность
- `tournament:start` - запуск
- `tournament:update` - обновление
- `tournament:player-joined` - игрок присоединился
- `tournament:player-left` - игрок покинул
- `tournament:player-ready` - готовность изменена
- `tournament:countdown` - обратный отсчет
- `tournament:start-game` - запуск игры

```
│   ├── page.tsx                    # Главная страница

│   ├── lobby/
│   │   ├── LobbyList.tsx           # Список лобби
│   │   └── LobbyRoom.tsx           # Комната лобби с игрой

├── lib/

```
