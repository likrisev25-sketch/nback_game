# 🏆 Онлайн-Турниры - Краткое Руководство

## Что было добавлено

### 1. Новая страница списка турниров `/tournaments`
- Просмотр всех доступных турниров со статусом "waiting"
- Создание новых турниров
- Присоединение к существующим турнирам
- Автоматическое обновление списка каждые 10 секунд

### 2. Страница турнира `/tournament/[id]`
- Просмотр участников турнира
- Статус готовности каждого игрока
- Кнопка "Готов/Не готов" для каждого игрока
- Запуск турнира (только для хоста)
- Обратный отсчет перед началом игры
- Real-time обновления через WebSocket

### 3. Backend API
- `GET /api/tournament/list` - Получить список турниров
- `POST /api/tournament/create` - Создать турнир
- `POST /api/tournament/join` - Присоединиться к турниру
- `GET /api/tournament/[id]` - Получить данные турнира

### 4. WebSocket Events для турниров
- `tournament:join` - Присоединение
- `tournament:leave` - Выход
- `tournament:ready` - Изменение статуса готовности
- `tournament:start` - Запуск турнира
- `tournament:update` - Обновление данных
- `tournament:player-joined` - Игрок присоединился
- `tournament:player-left` - Игрок покинул
- `tournament:player-ready` - Статус готовности изменен
- `tournament:countdown` - Обратный отсчет
- `tournament:start-game` - Запуск игры

### 5. База данных
- Таблица `tournaments` - основная информация о турнире
- Таблица `tournament_players` - игроки турнира

## Установка и запуск

### 1. Создание таблиц в базе данных

**Для SQLite:**
```bash
cd nback-game
npm run db:setup-tournament
```

**Для PostgreSQL (drizzle-kit):**
```bash
cd nback-game
npm run db:push
```

Или вручную выполните SQL из `drizzle/0001_tournament_tables.sql`

### 2. Запуск сервера с WebSocket

**Для разработки:**
```bash
npm run dev:socket
```

**Для продакшена:**
```bash
npm run build
npm run start:socket
```

### 3. Открыть приложение

Откройте браузер и перейдите на:
- `http://localhost:3000/tournaments` - список турниров
- `http://localhost:3000` - главная страница (ссылка на турниры)

## Использование

### Создание турнира

1. Зарегистрируйтесь/войдите в систему
2. Перейдите на `/tournaments`
3. Нажмите "Создать турнир"
4. Заполните форму:
   - Название турнира
   - N-значение (1-4)
   - Количество раундов (3/5/10)
   - Шагов в раунде (20/30/50)
   - Скорость (2500ms/2000ms/1500ms)
   - Минимум/максимум игроков (2-6)
   - Пароль (опционально)
5. Нажмите "Создать турнир"
6. Вы станете хостом и попадете на страницу турнира

### Присоединение к турниру

1. Перейдите на `/tournaments`
2. Найдите турнир в списке
3. Нажмите "Присоединиться"
4. Вы попадете на страницу турнира

### Запуск игры

1. Все игроки должны нажать "Готов"
2. Хост видит кнопку "Начать турнир" (активна когда все готовы и достаточно игроков)
3. Хост нажимает "Начать турнир"
4. Начинается обратный отсчет (5 секунд)
5. После отсчета игра запускается

## Важные замечания

1. **Требуется авторизация** - все пользователи должны быть авторизованы
2. **Socket.IO сервер** - необходим для real-time обновлений
3. **Минимум 2 игрока** - турнир нельзя запустить с одним игроком
4. **Только хост может запустить** - создатель турнира управляет запуском
5. **Все должны быть готовы** - турнир запускается только когда все игроки нажали "Готов"

## Тестирование

1. Откройте 2+ браузера (или режим инкогнито)
2. Войдите под разными аккаунтами
3. Один создает турнир, другие присоединяются
4. Проверьте real-time обновления:
   - Появление новых игроков
   - Изменение статуса готовности
   - Обратный отсчет
   - Запуск игры
