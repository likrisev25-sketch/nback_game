# Стартовая страница с обязательной регистрацией

## Что изменилось

### Было:
- Гостевой режим (можно играть без регистрации)
- Баннер с предложением войти
- Модальное окно авторизации по кнопке

### Стало:
- **Обязательная регистрация/вход** перед доступом к игре
- Красивая landing page с градиентным фоном
- Переключатель Вход/Регистрация
- Автоматический переход к игре после авторизации

## Архитектура

```
Пользователь заходит на /
         │
         ▼
┌─────────────────┐
│ sessionLoading? │
└────┬────────────┘
     │
     ├─ YES → Спиннер загрузки
     │
     └─ NO
        │
        ├─ !session?.user → LandingAuth (регистрация/вход)
        │
        └─ session?.user → GameMenu (игровое меню)
```

## Компоненты

### LandingAuth.tsx
Стартовая страница с формами авторизации.

**Особенности:**
- Градиентный фон с анимированными blob элементами
- Переключатель режимов (Вход/Регистрация)
- Интеграция с LoginForm и RegisterForm
- Авто-переход после успешной авторизации

```tsx
// Использование в page.tsx
if (!session?.user) {
  return <LandingAuth />;
}
```

## Flow авторизации

1. **Первый вход**
   ```
   / → LandingAuth → Регистрация → Better Auth создаёт user 
     → Cookies → useSession обновляется → GameMenu
   ```

2. **Повторный вход**
   ```
   / → useSession читает cookies → session есть → GameMenu
   ```

3. **Выход**
   ```
   GameMenu → LogoutButton → signOut() → cookies очищаются 
     → useSession null → LandingAuth
   ```

## Преимущества

| Было | Стало |
|------|-------|
| Можно играть без регистрации | Все игроки зарегистрированы |
| Анонимная статистика | Полная история игр |
| Нет мотивации создавать аккаунт | Аккаунт обязателен |
| Сложно отслеживать прогресс | Личный кабинет с статистикой |
| Нет лидерборда реальных игроков | Топ зарегистрированных игроков |

## Настройки

### Переключение режимов
```tsx
const [mode, setMode] = useState<'login' | 'register'>('register');
```

По умолчанию показывается **Регистрация** для привлечения новых пользователей.

### Анимации
- `animate-blob` — плавающие круги на фоне
- `backdrop-blur-lg` — размытие фона карточки
- `transition-all` — плавные переходы

## Интеграция

### 1. Добавить в page.tsx
```tsx
import { LandingAuth } from '@/components/auth/LandingAuth';

if (!session?.user) {
  return <LandingAuth />;
}
```

### 2. Обновить middleware
```tsx
// middleware.ts
const isProtectedRoute = nextUrl.pathname.startsWith('/profile');
// / больше не защищён — он сам обрабатывает авторизацию
```

### 3. Удалить гостевой режим
- Убрать поле ввода имени для гостей
- Убрать баннер "Гостевой режим"
- Все API вызовы используют `session.user`

## Тестирование

```bash
# 1. Запустить приложение
npm run dev

# 2. Открыть http://localhost:3000
# → Должна показать LandingAuth

# 3. Зарегистрироваться
# → Email + пароль → вход в игру

# 4. Выйти (LogoutButton в Header)
# → Снова LandingAuth

# 5. Войти с теми же данными
# → Автоматический вход (cookies)
```
