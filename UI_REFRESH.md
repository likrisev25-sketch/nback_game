### 4. Цветовая схема

| Элемент | Цвет |
|---------|------|
| Основной градиент | `pink-500 → purple-600` |
| Фон светлый | `indigo-50 → purple-50 → pink-50` |
| Фон тёмный | `gray-900 → purple-950 → pink-950` |
| Карточки | `white/70 + backdrop-blur` |
| Текст заголовков | Градиентный |
| Кнопки | Градиент + тень |
| Успех | `green-400 → emerald-500` |
| Ошибка | `red-400 → pink-500` |
| Предупреждение | `orange-400 → amber-500` |

### 5. Анимации

```css
/* Плавающая анимация */
@keyframes float {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-10px); }
}

/* Появление */
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(20px); }
  to { opacity: 1; transform: translateY(0); }
}

/* Пульсация градиента */
@keyframes gradientPulse {
  0%, 100% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
}
```
