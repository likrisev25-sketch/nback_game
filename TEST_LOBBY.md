# 🎮 Инструкция по проверке лобби и игры

## Быстрая проверка

### 1. Запуск сервера
```bash
cd nback-game
npm run dev
```

### 2. Откройте браузер
Перейдите на http://localhost:3000/lobbies

### 3. Создайте лобби
1. Нажмите "+ Создать лобби"
2. Заполните минимальную форму:
   - Название: "Мое тестовое лобби"
   - Минимум игроков: 2
   - Максимум игроков: 2
   - N-back: 2
   - Скорость: 2000ms
3. ✅ Поставьте галочку "Добавить бота"
4. Нажмите "Создать"

### 4. Проверка создания
Если всё работает:
- Появится модальное окно "Лобби создано! 🎉"
- Есть 3 кнопки:
  1. "Начать игру сразу" - переходит в лобби
  2. "Включить автозапуск" - переходит в лобби с автозапуском
  3. "Перейти в лобби без настроек" - просто в лобби

### 5. В лобби
После перехода в лобби вы увидите:
- Название лобби
- Настройки (N, скорость)
- Список игроков:
  - Вы (Хост)
  - Бот (Готов)
- Кнопки:
  - "Готов" - нажимаете
  - "Начать игру" - нажимаете (так как бот уже готов)

### 6. Запуск игры
После нажатия "Начать игру":
- Статус меняется на "in_progress"
- Показывается игра N-back
- Сетка 3x3
- Кнопка "Совпадение"
- Счёт правильных/ошибок

## Если не работает

### Ошибка создания лобби
Проверьте консоль браузера (F12):
```
🔵 [LobbyList] Создание лобби для пользователя: ...
🔵 [lobby/create] POST request received
🔵 [lobby/create] Request body: {...}
```

Проверьте консоль сервера:
```
✅ [db] Connected to local SQLite (nback.db)
🔵 [lobby/create] Creating lobby with userId: ...
✅ [lobby/create] Lobby created successfully: ...
```

### Лобби создаётся но не появляется в списке
Проверьте файл `nback.db` в корне проекта:
```bash
# Убедитесь что файл существует
ls nback.db
```

### Игра не запускается
В лобби проверьте:
1. Все игроки нажали "Готов" ✓
2. Хост нажал "Начать игру" ✓
3. Статус изменился на "in_progress" ✓

## Тестирование API вручную

### Создать лобби через curl:
```bash
curl -X POST http://localhost:3000/api/lobby/create \
  -H "Content-Type: application/json" \
  -d '{
    "gameId": "default",
    "name": "Тест",
    "nValue": 2,
    "baseSpeedMs": 2000,
    "minPlayers": 2,
    "maxPlayers": 2,
    "userName": "Игрок",
    "addBot": true,
    "botAccuracy": 80
  }'
```

### Получить список лобби:
```bash
curl http://localhost:3000/api/lobby/list
```

## Частые проблемы

### 1. "Требуется авторизация"
Решение: Зарегистрируйтесь или войдите через /login

### 2. "Не удалось создать лобби"
Решение: Проверьте консоль сервера на ошибки БД

### 3. Лобби создаётся но не появляется
Решение: Обновите страницу (F5)

### 4. Игра не запускается
Решение: Убедитесь что все игроки нажали "Готов"

### 5. Бот не появляется
Решение: Поставьте галочку "Добавить бота" при создании
